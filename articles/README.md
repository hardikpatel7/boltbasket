# Articles

Draft and published articles live here. Suggested file naming:

```
articles/
├── 2025-w01-meet-boltbasket.md             ← Week 1 Post 1 (long-form)
├── 2025-w01-etl-vs-elt-decision-framework.md ← Week 1 Post 2 (field note)
├── 2025-w02-the-two-revenues.md            ← Week 2 Post 1 (long-form)
└── ...
```

Status convention at the top of each file:

```markdown
---
status: draft | review | published
type: long-form | field-note
published_url: <Medium link if published>
arc: <which story arc, if any>
characters: <comma-separated list>
imperfection: <which schema imperfection drives the story>
---
```

When ready to draft, reference:
- `templates/long-form-post.md` or `templates/field-note.md`
- `style-guide.md`
- `bible/` for any context
- `schema/imperfections.md` to find the natural story setup
